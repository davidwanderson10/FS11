# O que é Node.js?
Definição: Node.js é um ambiente de execução JavaScript do lado do servidor, ou seja, permite rodar JavaScript fora do navegador.

## Por que usar?
Permite criar servidores, APIs e aplicações web escaláveis.
Muito usado em empresas modernas por ser rápido, flexível e ter uma grande comunidade.    
  
Conexão com a Stack:  
Antes: Usamos JS no front-end (React, por exemplo).
Agora: Usaremos JS para criar o back-end, que conversa com o front pelo HTTP.    
Depois: Faremos APIs REST completas, integraremos com bancos de dados, autenticação etc.

## O que é uma API?
API (Application Programming Interface): Uma ponte para que aplicações diferentes se comuniquem.  
API REST: Usa convenções HTTP (GET, POST, etc) para realizar operações (listar, criar, deletar, etc).  
Por que começar com uma rota?
Para entender o básico: como Node recebe requisições e responde.
