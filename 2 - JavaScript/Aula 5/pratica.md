## Exercícios práticos:

TODOS OS EXEMPLOS VOCÊ DEVE USAR HTML, CSS E JS EM ARQUIVOS SEPARADOS.

1. PROJETO IMC: COLOCAR CAMPOS PARA PESO E ALTURA NA PÁGINA E UM BOTÃO DE CALCULAR. AO CLICAR NO BOTÃO A PÁGINA DEVE INFORMAR O RESULTADO.

2. PROJETO MÉDIA: SOLICITE AS NOTAS DE UM USUÁRIO POR MEIO DOS CAMPOS NA TELA, COLOQUE UM BOTÃO PARA CALCULAR A MÉDIA E INFORME AO ALUNO SE ELE PASSOU, REPROVOU OU SE ESTÁ DE RECUPERAÇÃO.

3. CONVERSOR DE TEMPERATURAS: A TELA DEVERÁ MOSTRAR APENAS DOIS BOTÕES. UM PARA CONVERTER DE CELSIUS P/ FAHRENHEIT E OUTRO PARA CONVERTER DE CELSIUS PARA KELVIN. AO CLICAR EM UM DOS BOTÕES, DEVE APARECER UM TÍTULO LOGO ABAIXO, INFORMANDO QUAL É O TIPO DE CONVERSÃO E MAIS ABAIXO O CAMPO E O BOTÃO DE CALCULAR, O RESULTADO DEVE APARECER NO FINAL.
