## INTRUDUÇÃO À LÓGICA CONDICIONAL

Em programação, muitas vezes precisamos tomar decisões diferentes dependendo de certas condições. Para isso, usamos as estruturas condicionais.

Ex: "Se estiver chovendo, levo o guarda-chuva. Senão, vou sem."

### Estrutura if

```javascript

if (condição) {
    lógica
}

```

### Estrutura if - else

```javascript

if (condição) {
    lógica
} else {
    lógica
}

```

### Estrutura if - else if - else

```javascript

if (condição) {
    lógica
} else if (condição){
    lógica
} else {
    lógica
}
