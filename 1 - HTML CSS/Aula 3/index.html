O QUE É HTML?

A Linguagem de Marcação de Hipertexto (HTML) é uma linguagem de computador que compõe a maior parte das páginas da internet e dos aplicativos online.

Um hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação é composta por uma série de marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento.

O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos

PARA QUE SERVE O HTML?

O HTML (Linguagem de Marcação de Hipertexto) serve para estruturar o conteúdo de páginas da web, organizando elementos como textos, imagens, links e outros blocos de informação.

Funciona como o "esqueleto" de um site, definindo a estrutura e o significado do conteúdo para que navegadores, leitores de tela e mecanismos de busca possam interpretá-lo e exibi-lo. O HTML é a base da web, permitindo a criação de documentos eletrónicos e a interligação de informações através de hiperlinks.

QUAL A DIFERENÇA ENTRE HTML E LINGUAGENS DE PROGRAMAÇÃO?

A diferença principal é que HTML é uma linguagem de marcação para estruturar o conteúdo de uma página web, enquanto uma linguagem de programação é usada para criar algoritmos e executar lógica (como C++, Python ou JavaScript).

HTML não executa ações lógicas, não cria variáveis ou condicionais, apenas define como o conteúdo (texto, imagens, etc.) deve ser apresentado.

QUAIS AS FUNÇÕES DAS TAGS?

!DOCTYPE html?

A declaração !DOCTYPE html não é uma tag HTML, mas sim uma instrução que informa ao navegador qual é a versão do HTML utilizada e qual modo de renderização deve ser ativado, garantindo que a página seja exibida corretamente de acordo com os padrões da web.

Ela deve ser a primeira coisa em qualquer documento HTML e serve para evitar que o navegador entre em um modo de renderização especial, conhecido como "quirks mode

html lang="en/html

é um código HTML que especifica que o idioma principal de uma página da web é o inglês.

Este atributo é crucial para a acessibilidade, pois ajuda leitores de tela a pronunciarem o texto corretamente, orienta navegadores na exibição de caracteres e scripts adequados, e informa motores de busca sobre a linguagem do conteúdo da página.

head

define o cabeçalho de um documento, uma seção que contém metadados e links para outros recursos, mas que não é exibida diretamente na página web para o utilizador

meta charset="UTF-8

é uma meta tag HTML que define a codificação de caracteres para um documento web, instruindo o navegador a usar UTF-8 para exibir corretamente todos os caracteres

title

é um elemento HTML que define o título de uma página da web, aparecendo na aba do navegador e nos resultados de pesquisa

Body

define o corpo principal de uma página web, contendo todo o conteúdo visível e interativo, como textos, imagens, links, vídeos e formulários

h1

define o título principal e mais importante de uma página web, indicando o assunto central para utilizadores e motores de busca. É a maior heading da hierarquia (H1 a H6)

p

é usada para definir um parágrafo de texto

br

é um comando em HTML br que insere uma quebra de linha dentro de um texto, fazendo com que o texto seguinte apareça em uma nova linha, mas sem iniciar um novo parágrafo.
