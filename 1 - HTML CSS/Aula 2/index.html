Parágrafo


HTML, ou Linguagem de Marcação de Hipertexto, é a linguagem padrão usada para criar a estrutura de uma página web, definindo o conteúdo e sua organização com elementos como títulos, parágrafos, imagens e links.

Ao contrário de linguagens de programação, o HTML não é usado para criar funcionalidades, mas sim para estruturar o conteúdo de forma que navegadores possam interpretá-lo e exibi-lo corretamente.

É o "esqueleto" de uma página, sobre o qual outras tecnologias, como o CSS (para o visual) e o JavaScript (para a interatividade), adicionam detalhes e comportamento.
